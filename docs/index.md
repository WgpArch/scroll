# 🏠 Welcome to my Scroll Config

A highly customized Scroll compositor setup on Arch Linux. Scroll is a unique stacking/tiling Wayland compositor, and this configuration takes full advantage of its advanced layout modes, workspace scaling (overview), and deep tiling controls.

![Scroll Showcase](screenshots/screenshot_2026-05-24_19-22-59.png)

## ✨ Highlights
- **Advanced Layout Modes:** Deep integration with Scroll's modes (Jump, Filter, Resize, Floating, Align, Fit Size).
- **Workspace Scaling:** Bound to `Mod+Tab` for a beautiful overview of your workspaces.
- **Custom Waybar Dock:** A floating bottom bar with persistent icon-based workspaces and system metrics.
- **Smooth Animations:** Custom bezier curves for window moves, opens, and resizes.
